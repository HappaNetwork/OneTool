系统任务配置

开启本地回路能使监控运行更迅速。本地回路主要用于解决开启CDN之后访问IP和源站IP不一样而导致无法监控或监控慢的问题。
默认为空。设置密钥后，你需要在所有监控文件后面加上 ?cronkey=你的密钥 可防止监控文件被恶意执行
每次运行任务数是指在单个系统内，每运行一次监控文件所能够执行的任务数，建议50
用户申请补挂间隔的冷却时间，建议300
